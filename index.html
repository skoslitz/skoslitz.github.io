Markdown Syntax

Überschriften

Es stehen fünf Überschriftsebenen zur Verfügung, wobei H1 (Header 1) die größtmögliche darstellt.

Syntax: Ein Rautenzeichen # leitet eine Überschrift ein.

# Überschrift 1

## Überschrift 2

Hervorhebungen

Es sind kursiv und fette Hervorhebungen möglich.

Syntax kursiv: *kursiver Text*

Syntax fett: **fett geschriebener Text**

Listen

Listen dienen der besseren Ansicht von Aufzählungen. Es kann dabei zwischen unsortierten und sortierten Aufzählungen unterschieden werden.

Unsortierte Syntax:

+ Punkt 1 + Punkt 2

Sortierte Syntax:

1. Punkt 1 2. Punkt 2 3. Punkt 3 + Unterpunkt zur Punkt 3
